# KIV/SPOS - Správa počítačových sítí (Computer Network Management)

Tento repozitář obsahuje cvičení pro předmět "Správa počítačových sítí", zaměřený na výuku základů správy síťových služeb. Cílem je seznámit studenty s praktickými aspekty správy serverů a souvisejících technologií.

## Pokrytá témata:

*   **Základní správa Linux serveru:**
    *   SSH (Secure Shell)
    *   Fail2Ban (ochrana proti útokům hrubou silou)
    *   Firewall (iptables)
    *   Základní konfigurace a správa systému
*   **Správa disků:**
    *   Logical Volume Manager (LVM)
    *   RAID (Redundant Array of Independent Disks)
*   **DNS (Domain Name System)**:
    *   Konfigurace Bind serveru
*   **Webové servery:**
    *   Apache
    *   Nginx
*   **Databázové systémy:**
    *   MySQL
    *   PostgreSQL
*   **Emailové služby:**
    *   Postfix (MTA)
    *   Dovecot (IMAP/POP3)
*   **Síťové disky:**
    *   NFS (Network File System)
    *   Samba (CIFS pro Windows klienty)
*   **Windows Server:**
    *   (Téma k dopracování a upřesnění)
*   **Vysoká dostupnost:**
    *   Load balancing
    *   Další metody vysoké dostupnosti
*   **Bezpečnost serverů:**
    *   Portsentry
    *   RKHunter
    *   Další bezpečnostní nástroje
*   **Automatizace a orchestrace (volitelné):**
    *   Ansible
    *   Docker
    *   Traefik

Tento repozitář je v současné době ve fázi "strohé struktury cvičení".
